<h3 align="center">Platinum Events App</h3>

 <a name="introduction">Introduction</a>

Built on Next.js 14, the events application stands as a comprehensive, full-stack platform for managing events. It serves as a hub, spotlighting diverse events taking place globally. Featuring seamless payment processing through Stripe, have the capability to purchase tickets for any event or even initiate and manage your own events.

 <a name="tech-stack">Tech Stack</a>

- Node.js
- Next.js
- TypeScript
- TailwindCSS
- Stripe
- Zod
- React Hook Form
- Shadcn
- uploadthing
